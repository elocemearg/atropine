Display Setup

The Display Setup link in the sidebar takes you the page which administers the public-facing screen. Click the Open Display Window link at the top of this page to open the public-facing screen in a new browser window. You can then drag this new browser window onto a second screen, such as a projector or external monitor facing the players. Full-screen or resize this new browser window as necessary.

View selection - if in doubt, leave it on Auto

The menu on the Display Setup page shows you all the views you can choose from. If you want the least-effort solution, leave it on the default setting, "Auto".

If you want the public-facing window to show a particular screen, you can select it from the list of screen modes and click "Switch to this screen mode". The public-facing window will then show only that screen until you specify otherwise.

Default screen shape: Widescreen (16:9)

If you don't specify otherwise, Atropine assumes your public-facing display is on a Widescreen (16:9) device. Screen modes which show two different things at once will split the screen vertically. You can change this in Display Setup.

In Auto mode, Atropine uses the following rules to decide which screen to show:

  • If no players are entered yet, we display the Welcome screen
  • If there are players but no fixtures have been generated yet, we display the Player Check-In screen.
  • If the latest round has had fixtures generated but no scores have been filled in yet, we display the Fixtures screen.
  • Otherwise, we display the Standings / Results (Vertical) screen.

You can optionally have Auto mode show the Videprinter instead of the Results table when a round is in progress but not completed yet. The relevant checkbox can be found on the Display Setup page under the Auto screen mode.

Screen shape: Old (4:3)

If your public-facing display is an old-style 4:3 device, set the screen shape profile to Old (4:3) in Display Setup. Screen modes which display two things at once will split the screen horizontally.

In Auto mode in the 4:3 screen shape profile, Atropine uses the same rules as for Widescreen above, except that instead of the vertical Standings/Results screen we show the following:

  • If the latest round has had some but not all of its scores filled in, we display the Standings / Videprinter screen.
  • If the latest round has had all of its scores filled in, we display the Standings / Table Results screen.

Other modes

Auto mode never shows other screens such as the High Scores or Overachievers screens, so if you want to show those you need to switch to them manually using the Display Setup menu.
